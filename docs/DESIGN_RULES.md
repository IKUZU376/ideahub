# Design Rules

## Design Philosophy

IdeaHub is professional productivity software.

The interface should feel calm, modern and focused.

---

## Inspiration

- Linear
- GitHub
- Notion
- Vercel
- Raycast

---

## Visual Style

- Dark-first
- Minimal
- Premium
- Clean spacing
- Rounded corners
- Soft shadows
- High readability

---

## Colors

Primary

Purple / Indigo

Secondary

Neutral grayscale

Status

Green

Amber

Red

Blue

---

## Typography

Clear hierarchy.

Readable spacing.

Professional appearance.

---

## Components

Reuse existing components.

Avoid creating multiple versions of similar components.

---

## Layout

Responsive.

Collapsible sidebar.

Top navigation.

Consistent spacing.

---

## Animations

Subtle.

Fast.

Purposeful.

---

## General Rules

Never redesign working pages without reason.

Improve existing UI before replacing it.

Consistency is more important than novelty.
